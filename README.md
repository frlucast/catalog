# Catálogo Digital - Beauty Bloom

Catálogo de productos de belleza responsivo, dinámico y minimalista.

## Características
- **Carga dinámica**: Datos gestionados desde `data.json`.
- **Buscador y Filtros**: Filtrado por nombre y categorías en tiempo real.
- **Detalles en Modal**: Carrusel de imágenes y descripción detallada.
- **Integración WhatsApp**: Botón de compra directa por mensaje.
- **Paginación**: Carga progresiva de productos (20 en 20).

## Instalación
1. Descarga el proyecto.
2. Ejecuta un servidor local (ej. Live Server en VS Code).
3. Edita `data.json` para actualizar tus productos.
